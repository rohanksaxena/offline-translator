Features

Everything you need for
in-game translation

Built for game developers who need reliable, fast, offline translation without external dependencies or ongoing costs.

🌐
100% Offline

No internet connection required at runtime. Models run entirely on CPU using Intel oneDNN acceleration. Ship your game without worrying about API availability.

🔷
Blueprint Ready

Simple Blueprint nodes usable in any Blueprint. Two APIs available — Quick Translate for one-liners, and the full Subsystem for persistent state management.

⚡
Async Support

Non-blocking async translation nodes keep your game running smoothly. Translation happens on a background thread with callback on completion.

📝
Sentence & Paragraph Modes

Sentence mode for short NPC dialogue lines. Paragraph mode for multi-sentence text — splits, translates each sentence, then joins for more faithful output.

⚙️
Settings Panel

Full configuration via Project Settings → Plugins → Offline Translator. Language pair selection, model status indicator, model download management, CPU threads and beam size tuning.

💾
Model Management

Download models on demand via console command. Only download the language pairs you need. Models are staged with your game automatically for packaged builds.

🚀
Fast & Lightweight

Intel oneDNN INT8 quantization keeps models small and inference fast. First translation warms up compute kernels, subsequent translations are significantly faster.

📦
Packaged Build Support

Same API and performance in Editor, PIE, and fully packaged Shipping builds. Models are staged as loose files alongside your game content.

🔑
No API Keys

No accounts, no subscriptions, no per-request costs. Purchase once, use forever. Models are hosted on HuggingFace and downloaded directly into your project.

Blueprint Nodes

Clean, simple API

Two ways to use the plugin — the BP Library for quick one-liners, or the full Game Instance Subsystem for persistent state.

Initialize Translator
Load a language model into memory. Call once before translating. Async with On Ready callback.
Translate Sentence
Translate a single sentence synchronously. Returns translated text and timing.
Translate Sentence Async
Non-blocking sentence translation. Result via On Complete delegate.
Translate Paragraph
Translate multi-sentence text synchronously with better structure preservation.
Translate Paragraph Async
Non-blocking paragraph translation. Best for long text in gameplay.
Quick Translate
One-liner BP Library node. Translate without managing the subsystem directly.
Shutdown Translator
Unload the model and free memory when translation is no longer needed.
Is Translator Ready
Returns true if the model is loaded and ready to translate.
Is Language Pair Supported
Check if a given language pair has a model available.
Get Backend Name
Returns the active compute backend (e.g. oneDNN INT8_FLOAT32).
Get Offline Translator
Get the subsystem reference from anywhere in your Blueprint graph.
Get Supported Language Pairs
Returns an array of all supported language pair structs.

Quick Start

Up and running in minutes

Install from Fab, download a model, add two Blueprint nodes and start translating.

1
Install from Fab

Purchase and install from the Epic Games Fab Marketplace. Enable the plugin under Edit → Plugins → Offline Translator and restart the editor.

2
Add one line to DefaultGame.ini

Add to Config/DefaultGame.ini for packaged build model staging:


[/Script/UnrealEd.ProjectPackagingSettings]

+DirectoriesToAlwaysStageAsNonUFS=(Path="OfflineTranslatorModels")
3
Download a model

Open the Unreal console (backtick) and run:


OfflineTranslator.DownloadModel English German
4
Initialize and translate

In your Blueprint, call Get Offline Translator → Initialize Translator → wait for On Ready → call Translate Sentence Async. Done!
